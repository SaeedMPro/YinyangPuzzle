# YinyangPuzzle
This game is the final project of advanced programming course in University of Kurdistan.

YingYang Puzzle is a JavaFX game where the goal is to color a grid of buttons in such a way that two conditions are met: 

1. No 2x2 square should have the same color in all its cells.
2. All the white buttons should be connected to each other, and all the black buttons should be connected to each other. The two colors should not be interconnected.
## Features
- Randomly generates a grid of buttons with white and black colors.
- Allows the player to click on the buttons to change their colors.
- Provides visual feedback on whether the player's choices meet the game conditions.
- Displays a congratulatory message when the puzzle is successfully completed.
## Getting Started
1. Clone the repository: `git clone https://github.com/your-username/YingYang-Puzzle.git`
2. Open the project in your preferred Java IDE.
3. Build and run the project.
## Usage
- Left-click on a button to toggle its color between white and black.
- Right-click on a button to reset its color to gray.
- The status text at the top of the window displays messages about the player's choices and the game's outcome.
## Dependencies
- JavaFX
